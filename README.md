# UCH\_aircrafts

The repository contains R code used to generate main figures and tables for an upcoming publication on the microbiomes living in the vicinity of submerged aircrafts. All data used to generate plots is available in the data folder or the zenodo project [Microbial community structure and function on submerged aircraft wrecks](https://figshare.com/articles/dataset/dereplicated_MAGs/31653544).  The code can be viewed in the RMarkdown or pdf files.
